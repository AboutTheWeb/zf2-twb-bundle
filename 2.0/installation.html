TwbBundle, 1.0 - Installation / How to

Requirements

  • Zend Framework 2 (latest master)
  • Twitter Bootstrap (v3.0)

Install

Main Setup

By cloning project

  1. Clone this project into your ./vendor/ directory.
  2. (Optionnal) Clone the Twitter bootstrap project (v3.0) into your ./vendor/ directory.

With composer

  1. Add this project in your composer.json:

    "require": {
    "neilime/zf2-twb-bundle": "1.0"
}

  2. Now tell composer to download TwbBundle by running the command:

    $ php composer.phar update

Post installation

  1. Enabling it in your application.config.php file.

    return array(
    'modules' => array(
        // ...
        'TwbBundle',
    ),
    // ...
);

  2. Include Twitter Bootstrap assets

With AssetsBundle module (easy way)
  • Install the AssetsBundle module(2.1)
  • Install Twitter Bootstrap (v2.3.2)

  • Edit the application module configuration file module/Application/config/module.config.php, adding the configuration fragment below:

    return array(
    //...
     'asset_bundle' => array(
         'assets' => array(
             'less' => array('@zfRootPath/vendor/twitter/bootstrap/less/bootstrap.less')
         )
     ),
     //...
 );

  • Edit layout file module/Application/view/layout/layout.phtml, to render head scripts :

    //...    
echo $this->headScript();
//...
Manually
  • Copy bootstrap.css file (available on Twitter Bootstrap website) into your assets folder and add it in your head scripts
